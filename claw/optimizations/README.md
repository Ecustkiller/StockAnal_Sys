## 🔧 5 个核心模块详解

### ① `SectorOnedayDetector` — 板块一日游识别器

**原理**：首日爆发≥5只涨停的板块，根据次日涨停数衰减率+溢价表现识别"一日游"。

**阈值**：
- 衰减率 ≥ 70% 且 次日涨停<3 → **严重一日游 (扣 20 分)**
- 衰减率 ≥ 50% 或 次日平均涨跌 <-0.5% → **中度一日游 (扣 12 分)**
- 衰减率 < 30% → **主线延续 (加 3 分)**

**实盘验证**：
| 场景 | 结果 |
|---|---|
| 医药 16→1 (衰减 94%) | ✅ 严重一日游，扣20分 |
| 元器件 8→13 (反增) | ✅ 健康延续，加3分 |
| 化学制药 7→0 (衰减100%) | ✅ 严重一日游，扣20分 |

---

### ② `ZtContinuationRisk` — 连续涨停风险衰减器

**原理**：连板数越多，次日跌停风险指数级升高。参考北京炒家"3板以上不碰"+ 退学炒股"一鼓作气再而衰三而竭"。

**阶梯扣分**：
| 连板数 | 基础扣分 | 建议仓位 |
|---|---|---|
| 1 板（首板） | 0 | 100% |
| 2 板 | 3 | 70% |
| 3 板 | 8 | 40% |
| 4 板 | 15 | 20% |
| 5+ 板 | 25 | 0% |

**附加规则**：
- 5 日涨幅 > 40% → 额外扣 5 分
- **独苗龙头减免 50%**（4/17 圣阳 4 板仍可做的关键）

---

### ③ `EarlySealFactor` — 早盘封板时间因子

**原理**：封板时间越早 → 主动性越强 → 次日溢价越高。

**加减分表**：
| 封板时间 | 加分 | 典型股 |
|---|---|---|
| 09:30-09:35 | +8 | 秒板 |
| **09:35-09:45** | **+5** | 世运、可川 |
| 09:45-10:00 | +3 | 达实、东方电气 |
| 10:00-10:30 | +1 | 奥瑞德 |
| 10:30-13:00 | 0 | 劲旅 |
| **13:00-14:00** | **-2** | 洁美、奕东 |
| **14:00-14:30** | **-5** | 荣科 |
| 14:30 后 | -8 | 拉高出货嫌疑 |

**IC 验证**：本轮 10 只样本 Spearman ρ=0.299（正相关）。

---

### ④ `TopScoreReversal` — 高分反向修正器

**原理**：v3 高分票已充分定价 → 短期超额收益反而低。85-90 分的"黄金价值区"才是最佳区间。

**调整曲线**：
| v3 分值 | 调整 | 含义 |
|---|---|---|
| 95+ | **-8** | ⚠️ 超高分，强抑制 |
| 90-95 | -3 | 🟡 高分，轻抑 |
| **85-90** | **+2** | 🏆 **黄金价值区** |
| 75-85 | +1 | 次优区 |
| <75 | 0 | 不动 |

**4/17 回归验证**：
| 指标 | v3 | v4 | 改善 |
|---|---|---|---|
| TOP-6 平均涨幅 | +0.67% | +1.38% | +0.72% |
| TOP-8 平均涨幅 | +1.34% | +2.16% | +0.83% |
| TOP-10 平均涨幅 | +1.77% | +2.76% | +0.99% |
| 中钨高新(+12.8%) 进 TOP10 | ❌ | ✅ |  |
| 联合精密(+7.9%) 进 TOP10 | ❌ | ✅ |  |

---

### ⑤ `SectorNextDayVerify` — 板块次日验证器

**原理**：状态机管理板块生命周期，避免"首日爆发直接升级为主线"的系统性误判。

**状态流转**：
```
无信号 ──(当日涨停≥5)──> 候选主线 ──(次日验证通过)──> 确认主线 ✅ 可做
                            │
                            └──(次日验证失败)──> 一日游 ❌ 回避
```

**次日验证 3 条规则（任 2 条通过即升级）**：
1. **数量验证**：次日涨停/首日 ≥ 50%
2. **溢价验证**：首日涨停票次日平均涨跌 ≥ +0.5%
3. **龙头验证**：首日龙头次日未跌停且未暴跌<-3%

**4/15→4/16 回放**：
- 医药（16→1，均涨-1.7%，龙头跌停-10%）→ **一日游** ✅
- 元器件（8→13，均涨+2.5%，龙头+8%）→ **确认主线** ✅

## 🚀 使用方式

### 方式 1：一键集成（推荐）

```python
from optimizations import apply_v4_optimizations

# v3 评分后的 DataFrame（需含 industry, is_zt, seal_time, consecutive_zt, total 列）
df_v3 = score_system.run()

# 市场上下文
market_ctx = {
    'prev_sectors_zt': {'元器件': 8, '医药': 16, ...},
    'curr_sectors_zt': {'元器件': 13, '医药': 1, ...},
    'prev_zt_next_perf': {'002294.SZ': -10.0, ...},
}

# 一键应用 v4
df_v4 = apply_v4_optimizations(df_v3, market_ctx)

# df_v4 新增列：oneday_penalty, zt_risk_penalty, early_seal_bonus,
#              reversal_adj, v4_score, final_rank
print(df_v4[['name', 'total', 'v4_score', 'final_rank']].head(10))
```
